Aveti Learning App

I developed the Aveti Learning app from scratch, utilizing MVVM, Jetpack components, State Flow, Coroutines, and Dependency Injection, creating an app that serves over 15,000 students, including 300+ visually impaired users. I implemented a download manager with Media3 ExoPlayer, enabling offline access to adaptive streams (240p, 480p, 1080p), and introduced an offline-first architecture with on-device encryption and decryption for managing exercises, exams, and videos, secured via Android Keystore. To enhance accessibility, I integrated TalkBack support, ensuring visually impaired students can use the app independently. I also reduced server costs by 65% through ETag-based API caching with Room DB, efficiently handling over 1,000 daily API requests with zero downtime. Additionally, I designed a cost-effective DRM alternative, achieving 95% security while keeping content affordable, and integrated user-choice billing with Google Play and Razorpay, achieving a 100% transaction success rate. Finally, I built a data injection pipeline using HTML/CSS/Perseus files to deliver interactive exercises, enriching the learning experience.

Shikhya App

I enhanced the Smart Class App by solving critical functionality and security issues. To address storage restrictions on Android 11+ devices, which prevented USB drive access and disrupted educators' workflows, I implemented the Storage Access Framework (SAF), restoring seamless read/write capabilities and ensuring uninterrupted access to educational content. Additionally, I tackled a security vulnerability where temporary in-memory decryption of videos exposed the app to attacks. By integrating libmedia to stream videos directly from USB drives, I eliminated in-memory decryption, reducing memory-based attacks by 50%. These solutions, leveraging Android, SAF, and libmedia, significantly improved the app's compatibility, security, and user experience.
